LIST

there are three type of list :-

  • Ordered list
  • Unordered list
  • Description list

1=>ordered list

The HTML <ol> tag defines an ordered list. An ordered list can be numerical or alphabetical.

2=> Unordered list

The HTML <ul> tag defines an unordered (bulleted) list. The list items will be marked with bullets (small black circles) by default:

3=> Description list

A description list is a list of terms, with a description of each term.

The <dl> tag defines the description list, the <dt> tag defines the term (name), and the <p> tag describes each term
